* Setting up:
1. Got setting done with URP.
2. With BASE sprite, go to Sprite Editor
3. In BASE's Sprite Editor, go to Sprite Editor/Secondary Texture
4. Add + new item:
Name: Typed in "_NormalMap"
Texture: Drag or chosse NORMAL sprite
5. Apply.

* In scene:
1. Drag an BASE sprite into scene.
2. Create a new Light 2D (Spot Light 2D).
3. In URP ver 14.0.9, Normal Map can be turn on in Quality/Fast
